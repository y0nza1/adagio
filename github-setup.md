# GITHUB

### SETUP
1. Install/Ensure Git is up to date.
  * `sudo apt update`
  * `sudo apt install git-all`
2. Configure username and e-mail address for commits.
  * `git config --global user.name "YOUR NAME"`
  * `git config --global user.email "your_email@dmail"`
3. Install Github CLI (gh).
  * `sudo apt install gh`
4. Authenticate with GitHub.
  * `gh auth login`
5. Follow the instructions.
